# Docker & Flask App Lab

Instruqt 2.0 Lab — Docker + Flask App running on port 5000.

## What this lab does

- Spins up an Ubuntu 22.04 container with Docker installed via startup script
- Pulls and runs `shivtushal/git-lab:python-app-1.0` on port 5000
- Learner verifies Docker is running via `docker ps`
- Service tab exposes port 5000 in the browser
